# HXPFC_MATLAB
The included code contains MATLAB code for the Hydrodynamic Structural Phase Field Crystal Method (HXPFC). If this code is utilized, altered, or modified, please cite our paper: Crystals 2025, 15(2), 157; https://doi.org/10.3390/cryst15020157 .
